# ACS WILDA Team 7 – Customer Churn Analysis Project

## Project Overview
This project focuses on analysing customer churn using data preparation, clustering analysis, and predictive modelling techniques.

The objective is to identify key customer segments and develop insights to support churn prediction and business decision-making.

## Project Stages
- Stage 1: Project planning and setup
- Stage 2: Data Preprocessing & Preparation | Clustering Analysis
- Stage 3: Predictive modelling | Data Visualisation and Final Report
